Διαχειριστικό endpoint για την διαγραφή όλων των καταγεγραμμένων απαντήσεων  στις ερωτήσεις του ερωτηματολογίου με αναγνωριστικό questionnaireID.
Resource URL
http://localhost:9103/intelliq_api/admin/resetq/{questionnaireID}
Parameters
questionnaireID (Path)
required

Το αναγνωριστικό του ερωτηματολογίου προς διαγραφή.

Type: int
Example Values: 203
Curl Example
$ curl -X POST -d '' "http://localhost:9103/intelliq_api/admin/resetq/{questionnaireID}"
