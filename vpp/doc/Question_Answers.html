Επιστρέφει JSON object με όλες τις απαντήσεις από όλα τα γεγονότα απάντησης (sessions) που δότηκαν στην ερώτηση με αναγνωριστικό questionID στο ερωτηματολόγιο questionnaireID, ταξινομημένες ως προς τη χρονική σειρά με την οποία δόθηκαν.
Resource URL
http://localhost:9103/inteliq_api/getquestionanswers/{questionnaireID}/{questionID}
Parameters
questionnaireID (Path)
required
Το αναγνωριστικό του ερωτηματολογίου
Type: string
questionID (Path)
required
Το αναγνωριστικό της ερώτησης
Type: int
Curl Example
$ curl -X GET "http://localhost:9103/inteliq_api/getquestionanswers/{questionnaireID}/{questionID}"
